## Promotional Text

## Description
CodexIPhone is an iOS client for teams and individuals who run their own relay server and desktop connector.

Key features:
- Connect to a custom Relay URL with Bearer token
- Monitor connector/runner status in real time
- Create threads, send messages, and review task flow
- Manage approvals and check runtime health from mobile

This app does not provide a hosted backend service. A self-hosted relay deployment is required.
