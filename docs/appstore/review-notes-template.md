# App Review Notes Template

## How to access the app
1. Launch app.
2. In onboarding page, input Relay Base URL and Bearer token.
3. Tap `Save and Start`.

## Test environment
- Relay URL: <RELAY_BASE_URL>
- Bearer token: <RELAY_TOKEN>
- Workspace: default

## What reviewers can verify
- Status page loads connector/runner heartbeat
- Thread list loads from relay
- Message send/receive works in selected thread
- Approvals page loads pending approvals

## Encryption declaration
`ITSAppUsesNonExemptEncryption` is set to `NO`.
