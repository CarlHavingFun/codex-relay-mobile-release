# App Store 元数据模板（zh-CN）

## App 名称
Agent Relay Mobile

## 副标题
自建 Relay 的移动端客户端

## 推广文本
在 iPhone 上连接并管理你自建的 Relay 服务。

## 描述
Agent Relay Mobile 是用于自建 Relay 服务与桌面 Connector 的 iOS 客户端。

主要能力：
- 通过 Relay URL + Bearer Token 接入你的服务
- 实时查看 connector/runner 状态
- 创建线程、发送消息、跟踪任务流
- 在手机侧查看审批与运行健康状态

本应用不提供托管后端，需用户自行部署 Relay 服务。
